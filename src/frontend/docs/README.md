# EOATravelDeals Documentation

Welcome to the EOATravelDeals documentation. This directory contains guides and resources for developers and users.

## Available Guides

### [Android Installation Guide](./android.md)
Learn how to install and run EOATravelDeals on Android devices:
- **Progressive Web App (PWA)**: Install directly from Chrome browser (recommended)
- **Debug APK**: Build and sideload a native Android wrapper for testing
- **Internet Identity**: Troubleshooting authentication on Android
- **Prerequisites**: Required tools and environment setup for APK builds
